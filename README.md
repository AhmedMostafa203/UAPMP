> **Empowering universities with seamless academic management and productivity tools.**
> UAPMP is a full-stack platform connecting students, instructors, and administrators for a smarter, more efficient academic experience.

---

## 🚀 Key Features

- **Secure Authentication** (JWT, Google OAuth)
- **Admin Dashboard** for user, class, and announcement management
- **Class & Material Management**
- **Announcements** with scheduling and comments
- **Assignments & Submissions** workflow
- **Attendance Tracking** with export
- **AI Productivity Tools**
- **Responsive, Modern UI**

## 📦 Installation & Setup

1. **Clone the repository:**
   ```bash
   git clone <repository-url>
   cd UAPMP
   ```
2. **Backend Setup:**
   ```bash
   cd backend
   npm install
   # Configure environment variables as needed
   npm start
   ```
3. **Frontend Setup:**
   ```bash
   cd ../frontend
   npm install # If using a build system
   # Or open index.html directly in your browser
   ```
4. **Configuration:**
   - Update `config/` files for backend settings.
   - Update `frontend/config.js` for frontend API endpoints.
